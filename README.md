## Descripción del proyecto

Este proyecto consiste en el desarrollo de una **página web de perfil personal** utilizando **HTML5 y CSS3**.
La página incluye tres secciones principales:

* Perfil del estudiante
* Lista de habilidades
* Formulario de contacto

En el desarrollo se aplican conceptos de **CSS3** como selectores avanzados, uso del **Box Model con `box-sizing: border-box`**, posicionamiento con **`fixed`, `relative` y `absolute`**, además de estilos accesibles para formularios.

---

## Instrucciones para ejecutar el proyecto

1. Clonar o descargar el repositorio.
2. Abrir la carpeta del proyecto en **Visual Studio Code**.
3. Abrir el archivo **index.html**.
4. Ejecutar el archivo utilizando la extensión **Live Server**.

La página se abrirá automáticamente en el navegador **Google Chrome**.

---

## Visualización

La página se visualiza correctamente en el navegador **Google Chrome** en diferentes resoluciones, incluyendo:

* **1280px** (pantalla de computador)
* **375px** (pantalla móvil)
